# 大连理工大学线性代数上机课

![MATLAB](https://img.shields.io/badge/MATLAB-R2020b%2B-blue)

这个仓库包含了使用MATLAB进行的线性代数上机课作业。

## 目录结构

```
.
├── Class1/                # 第一类练习
│   ├── hw1.m              # MATLAB源代码文件
│   └── report.md          # 练习报告
└── README.md              # 本说明文件
```

## 内容概览

本项目目前包含以下练习：

- 基础矩阵运算练习（Class1）
  - 矩阵加减法
  - 矩阵乘法及相关性质验证
  - 行列式、逆矩阵和秩的计算
  - 矩阵方程求解


## 使用方法

1. 克隆或下载此仓库到本地
2. 在MATLAB中打开相应目录下的 `.m` 文件
3. 运行脚本查看结果

或者，在MATLAB命令窗口中执行相应命令。

## 各类练习详情

每个练习目录下都有相应的源代码和报告文件：
- `.m` 文件包含可执行的MATLAB代码
- `report.md` 文件包含详细的过程和结果说明

请查看各子目录以获取具体信息。
